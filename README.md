# CareerOS Germany

CareerOS Germany is a Next.js portfolio project for international students in Germany who need to track job applications, student work-hour limits, CV readability, and skill gaps.

The project is currently in the visual prototype stage:

- `/` cinematic public hero screen
- `/dashboard` full-screen dashboard mock
- planning docs in `implementation-plan.md`

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Current stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Lucide icons
